# game status

## Description

Shows the status of the current game.

## Help Output

```
USAGE
  $ shipthis game status [-g <value>]

FLAGS
  -g, --gameId=<value>  The ID of the game

DESCRIPTION
  Shows the status of the current game.

EXAMPLES
  $ shipthis game status

  $ shipthis game status --gameId 0c179fc4
```
