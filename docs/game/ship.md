## `shipthis game ship`

Builds the app (for all platforms with valid credentials) and ships it to the stores

```

FLAGS
  -g, --gameId=<value>  The ID of the game

DESCRIPTION
  Builds the app (for all platforms with valid credentials) and ships it to the stores

EXAMPLES
  $ shipthis game ship
```
